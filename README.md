# Simon Game

This is a simple Simon Game that I built while learning JavaScript and jQuery.  
The game generates a random sequence of colors, and the player has to repeat the same pattern correctly.

## Built With
- HTML
- CSS
- JavaScript
- jQuery

## What I Learned
While building this project, I practiced:
- DOM manipulation
- Event listeners
- jQuery animations and effects
- Arrays and functions in JavaScript
- Basic game logic

## Features
- Random sequence generation
- Button animations and sound effects
- Game over detection
- Keyboard start functionality

## Future Improvements
Some things I want to add later:
- Score tracking
- Difficulty levels
- Better mobile support
- Restart button

## Run the Project
Download or clone the repository and open `index.html` in your browser.
